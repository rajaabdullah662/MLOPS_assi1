# Diabetes Prediction: From Data to Deployment

## Project Description

This project uses a diabetes medical dataset to clean data, perform exploratory data analysis, train machine learning classification models, and deploy the best model using FastAPI.

## Dataset

The dataset used is `diabetes_unclean.csv`.

## Steps Performed

1. Data cleaning using Pandas
2. Exploratory data analysis using Matplotlib and Seaborn
3. Model training using Scikit-learn
4. Model evaluation using accuracy, precision, recall, and F1-score
5. Best model saving using Joblib
6. FastAPI deployment
7. API testing using cURL
